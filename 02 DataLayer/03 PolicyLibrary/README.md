### 政策类型分类

| 类型 | 说明 | 典型政策 |
|------|------|----------|
| 法律 | 全国人大制定的法律 | 《民法典》《票据法》 |
| 行政法规 | 国务院制定的法规 | 《票据管理实施办法》 |
| 部门规章 | 金融监管部门制定的规章 | 银保监会、央行发布的规章 |
| 规范性文件 | 监管部门发布的指导意见 | 《关于规范商业承兑汇票信息披露的公告》 |
| 行业标准 | 行业协会制定的标准 | 票交所业务规则 |

### 监管机构

- **国务院**：最高行政机关
- **人民银行**：货币政策、支付结算
- **银保监会**：银行、保险监管
- **证监会**：证券、基金监管
- **外汇管理局**：外汇管理
- **票交所**：票据业务规则
- **行业协会**：行业自律规则
