### 话术类型分类

| 类型 | 说明 | 场景 |
|------|------|------|
| 开场话术 | 破冰、引起兴趣 | 初次接触 |
| 产品介绍 | 产品核心卖点 | 产品讲解 |
| 异议处理 | 应对客户疑虑 | 沟通过程 |
| 成交话术 | 促进签单 | 谈判后期 |
| 维护话术 | 客户维护 | 售后服务 |

### 切入角度分类

| 角度 | 说明 | 适用场景 |
|------|------|----------|
| 痛点切入 | 直击客户痛点 | 客户有明确痛点 |
| 利益切入 | 强调收益价值 | 客户关注收益 |
| 案例切入 | 同行业成功案例 | 客户需要验证 |
| 政策切入 | 政策红利/合规 | 政策敏感型客户 |
| 风险切入 | 规避风险 | 风险敏感型客户 |
| 时效切入 | 时间紧迫性 | 有时间压力的客户 |

## 话术模板示例

### 示例1：反向保理 - 痛点切入（制造业上游供应商）

**开场话术**：
> "王总，我了解到贵公司主要给XX品牌做电子元器件供应，年营收5000万左右。现在很多像您这样的供应商都面临一个问题——应收账款周期长，90-120天才能回款，订单多了反而资金紧张。我们有个方案可以帮您快速回笼资金，最快3天就能拿到钱，您方便了解一下吗？"

**产品介绍话术**：
> "这个方案叫'反向保理'，简单说就是利用您的核心企业（XX品牌）的信用，帮您提前拿到货款。好处是：第一，不用您提供抵押物；第二，利率比普通贷款低；第三，额度灵活，订单多了额度也跟着涨。"

**异议处理话术**：
```
客户疑虑：利率会不会很高？
应对话术：王总，这个您放心。因为是利用核心企业信用，利率通常比您自己贷款低很多，大概在4-6%左右，而且按天计息，用几天算几天，比您资金紧张时去借高利贷划算多了。

客户疑虑：手续会不会很复杂？
应对话术：手续很简单，您只需要提供跟核心企业的合同和发票，我们帮您对接，线上就能办，最快3天放款。

客户疑虑：核心企业会知道吗？
应对话术：核心企业会知道，但这其实是好事。说明您在跟他们合作，而且他们也能通过这个方式帮助供应商，对他们供应链管理也有好处。
```

**成交话术**：
> "王总，要不这样，您先试一单，看看效果。额度批下来后，随时可以用，不用不产生费用。您看这周什么时候方便，我带资料过去给您详细讲讲？"

### 示例2：商票贴现 - 利益切入（商贸流通企业）

**开场话术**：
> "李总，您手上现在有XX集团开出的商票吗？我这边有个方案可以帮您把这些商票快速变现，而且利率比市场上低不少。您现在手上有多少商票？"

**产品介绍话术**：
> "我们这个商票贴现产品，最大的优势是：第一，线上操作，不用跑银行；第二，利率透明，没有额外费用；第三，最快当天到账。而且XX集团的商票在我们行有授信额度，贴现利率能到3.5%左右。"

## 话术优化机制

### 话术迭代
1. 记录使用效果（成功率）
2. 分析失败原因
3. 优化话术内容
4. A/B测试不同版本

### 话术禁忌
- ❌ 不要夸大收益
- ❌ 不要隐瞒风险
- ❌ 不要承诺无法做到的事
- ❌ 不要使用专业术语堆砌
- ❌ 不要贬低竞争对手
