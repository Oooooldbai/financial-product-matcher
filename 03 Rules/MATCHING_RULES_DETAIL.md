# 匹配规则细化 - 产品×客户具体匹配逻辑

> 基于产品库（91个+产品）和客户库（113家核心企业）的具体匹配规则

---

## 匹配规则细化方法

### 方法论
```
产品库特征 × 客户库特征 = 具体匹配规则
```

### 维度对应

| 产品库维度 | 客户库维度 | 匹配逻辑 |
|-----------|-----------|---------|
| 目标客户类型 | 企业性质 | 类型匹配 |
| 准入条件 | 规模/财务指标 | 门槛匹配 |
| 产品特色 | 供应链位置 | 场景匹配 |
| 账期特点 | 平均账期 | 时间匹配 |
| 核心企业关联 | 核心企业列表 | 关系匹配 |

## 三、匹配度计算细则

### 计算公式

```
总分 = 痛点匹配度×0.30
     + 需求匹配度×0.25
     + 准入条件匹配×0.20
     + 利率优势×0.10
     + 核心企业关联×0.10
     + 行业经验×0.05
```

### 具体打分细则

#### 1. 痛点匹配度（30分）

| 痛点类型 | 匹配度 | 说明 |
|---------|-------|------|
| 资金紧张 | 10分 | 产品能快速提供资金 |
| 账期长 | 8分 | 产品能延长账期或让供应商T+0到账 |
| 无抵押物 | 7分 | 产品无需抵押 |
| 融资成本高 | 5分 | 产品融资成本低 |

#### 2. 需求匹配度（25分）

| 需求类型 | 匹配度 | 说明 |
|---------|-------|------|
| 快速融资 | 10分 | 产品T+0到账 |
| 稳定供应链 | 8分 | 产品能稳定供应链关系 |
| 降低成本 | 7分 | 产品能降低融资成本 |

#### 3. 准入条件匹配（20分）

| 条件类型 | 匹配度 | 说明 |
|---------|-------|------|
| 企业性质匹配 | 5分 | 客户类型符合产品要求 |
| 规模匹配 | 5分 | 年营收/年应付符合要求 |
| 行业匹配 | 5分 | 行业符合产品定位 |
| 账期匹配 | 5分 | 平均账期符合产品要求 |

#### 4. 利率优势（10分）

```
利率优势 = (市场平均利率 - 产品利率) × 10
         最低0分，最高10分
```

#### 5. 核心企业关联（10分）

| 关联类型 | 匹配度 | 说明 |
|---------|-------|------|
| 有授信 | 10分 | 核心企业有授信额度 |
| 有合作历史 | 5分 | 核心企业有合作记录 |
| 无关联 | 0分 | 无关联 |

#### 6. 行业经验（5分）

| 经验类型 | 匹配度 | 说明 |
|---------|-------|------|
| 有同行业案例 | 5分 | 银行有同行业成功案例 |
